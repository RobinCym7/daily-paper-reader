## 摘要
点云语义分割需要能够捕捉细粒度局部几何和广泛全局场景结构的架构。基于Transformer的网络通过专注于详细的局部特征聚合展示了强大的性能；然而，全局上下文主要通过跨编码器-解码器阶段的跳跃连接来传递，我们认为这对于完整场景理解是不够的。我们假设，通过可学习的全局特征提取模块增强跳跃连接，使网络在深入局部细节之前获取场景级知识，从而得到更丰富且更具上下文依据的表示。为此，我们提出了带小波神经算子的点Transformer（PT-WNO），它在点云Transformer骨干的跳跃连接旁集成了一个共享的小波神经算子（WNO）分支。在每次编码器-解码器过渡时，点特征被投影到一个密集的3D体积网格上，WNO通过可学习的小波分解和重构捕捉多尺度全局谱上下文。这些全局特征通过轻量级适配器融合回网络，补充而非取代现有的跳跃连接。在四个大规模3D点云基准上的实验证明了PT-WNO的有效性。在S3DIS（Area 5）上，PT-WNO达到71.59% mIoU，比Point Transformer v3（PTv3）基线高出1.03个百分点。在DALES上，它达到81.05% mIoU（比基线高1.47）。在ScanNet v2上，PT-WNO获得76.19% mIoU，与基线（76.36%）保持竞争力。

## Abstract
Point cloud semantic segmentation requires architectures that capture both fine-grained local geometry and broad global scene structure. Transformer-based networks have demonstrated strong performance by focusing on detailed local feature aggregation; however, global context is conveyed primarily through skip connections across encoder-decoder stages, which we argue is insufficient for full scene understanding. We hypothesize that augmenting skip connections with a learnable global feature extraction module allows the network to acquire scene-level knowledge before descending into local detail, leading to richer and more contextually grounded representations. To this end, we propose Point Transformer with Wavelet Neural Operato (PT-WNO), which integrates a shared Wavelet Neural Operator (WNO) branch alongside the skip connections of a point cloud transformer backbone. At each encoder-decoder transition, point features are projected onto a dense 3D volumetric grid where the WNO captures multi-scale global spectral context through learnable wavelet decomposition and reconstruction. These global features are fused back into the network via lightweight adapters, complementing rather than replacing the existing skip connections. Experiments on four large-scale 3D point cloud benchmarks demonstrate the effectiveness of PT-WNO. On S3DIS (Area 5), PT-WNO achieves 71.59% mIoU, outperforming the Point Transformer v3 (PTv3) baseline by +1.03 points. On DALES it achieves 81.05% mIoU (+1.47 over the baseline). On ScanNet~v2, PT-WNO obtains 76.19% mIoU, remaining competitive with the baseline (76.36%).

- **研究动机**：假设在跳跃连接中引入一个可学习的、显式的全局特征提取模块，能让网络在深入局部细节之前先获取场景级先验知识，从而生成更丰富、更具上下文基础的表示。
- **整体含义**：提出将小波神经算子（WNO）作为全局上下文建模工具，与点云 Transformer 主干网络融合，在不牺牲效率的前提下增强长程依赖和多尺度结构理解能力，尤其适用于室内大场景和彩色信息缺失的室外 LiDAR 数据。

## 2. 方法论

- **整体框架（PT-WNO）**：
  - 以序列化点 Transformer（PTv3）为骨干，采用 U-Net 风格的编码器-解码器结构，利用空间填充曲线（Hilbert/Z-order）将无序点云转为 1D 序列，实现高效的窗式注意力（windowed attention）。
  - 在编码器-解码器的跳跃连接旁，并联一个**共享的小波神经算子分支**，在若干编码器阶段对点特征进行全局、多尺度融合。

- **关键技术细节**：
  1. **网格投影（Grid Projection）**：将编码器某一阶段的点特征通过轻量 MLP 投影到统一的维度 \(D\)，然后利用线性映射和均值池化将特征分配到固定的 3D 体素网格（resolution \(64^3\)）上，形成稠密张量 \(\mathbf{U}^{(s)}\)。
  2. **相对位置编码（Relative Position Encoding）**：针对体素化后空间信息的丢失，计算每个点相对于该阶段点云质心的归一化位移及半径，通过小型 MLP 生成位置偏置 \(\mathbf{e}_i\)，在体素化之前逐点加到特征上，使全局算子感知几何布局。
  3. **小波神经算子（Wavelet Neural Operator）**：
     - 本质上学习一个在函数空间上的积分算子，通过在小波域对角化核来实现多尺度特征混合。
     - 实际实现中，由于固定小波滤波器库不适用于 3D 不规则点云，改用**可训练的深度可分离步长卷积**（分析路径）和**转置卷积**（合成路径）来模拟离散小波变换与逆变换，保留多级分解与重建的归纳偏置，并配有残差旁路（bypass term）。
     - 形式化为 \(\mathbf{V}^{(s)} = \mathcal{K}_\theta(\tilde{\mathbf{U}}^{(s)})= \text{IDWT}\big(R_\theta \odot \text{DWT}(\tilde{\mathbf{U}}^{(s)})\big) + B(\tilde{\mathbf{U}}^{(s)})\)。
  4. **网格到点提升（Grid-to-Point Lifting）**：经过 WNO 得到的栅格特征 \(\mathbf{V}^{(s)}\)，通过查找对应体素索引取出，再用 MLP 投影回骨干的通道维度。
  5. **多阶段特征融合**：将 WNO 输出的全局特征 \(\mathbf{g}_i'\) 与骨干的局部特征 \(\mathbf{h}_i\) 在编码器各阶段进行串联、线性投影、LayerNorm 和 GELU 激活，替代原始特征继续向前传播，并经由跳跃连接传入解码器。
  6. **训练损失**：采用交叉熵损失与 Lovász-Softmax 损失的组合，端到端优化整个网络，无需额外的辅助分支损失。

## 6. 主要结论与发现

- **全局提升**：PT-WNO 在 S3DIS Area 5 上 mIoU 达 71.59%（提升 1.03 点），在 DALES 上达到 81.05%（提升 1.47 点），显著优于 PTv3 基线；在 ScanNet v2 上与基线持平（76.19% vs 76.36%），差异在运行方差内。
- **几何敏感性与数据适应性**：小波算子对几何轮廓明显、颜色信息贫乏的场景（如 DALES 室外 LiDAR）或特定几何类别（如 S3DIS 中的柱子、门、沙发）提升明显；在已有密集表面和丰富颜色纹理的 ScanNet 中，增益微小，说明 WNO 是一种几何补充而非替代。
- **参数开销极小**：共享 WNO 分支仅增加约 1.6M 参数（从 46.2M 到 47.8M，提升 3.5%），展现出高性价比的全局建模能力。
- **定性改善**：预测结果的空间一致性更好，边界更清晰，孤立错分减少，在室内大房间和野外大范围地形中均表现出更强的布局理解。

## 7. 优点

- **创新性架构**：首次将小波神经算子系统性地融入点云 Transformer，将全局上下文视为显式的连续算子，而非仅由深层局部注意力堆叠涌现。
- **高效且轻量**：全局分支参数共享、计算在小尺寸网格上进行，几乎不影响原网络的推理效率，且大幅提升了特定场景的性能。
- **对几何信息的鲁棒性**：去除颜色特征后性能几乎不衰减，证明模块能有效编码纯几何线索，对缺少 RGB 的 LiDAR 数据极为友好。
- **灵活融合机制**：相对位置编码和多阶段融合设计，使全局算子既能理解“特征是什么”，也能知道“特征在场景中何处”。
- **可复现的严格对比**：专门复现了 PTv3 基线，消除了环境、超参数带来的不公平提升。

## 摘要
传统的独热编码常导致模型校准不良，在攻击下过于自信，并使基于熵的检测算法失效。先前在图像分类中的研究表明，Hadamard编码的输出表示可以提高对抗鲁棒性。然而，将Hadamard编码集成到语义分割中的尝试，在平均交并比性能上远落后于最先进模型。对于目标检测，此类输出编码尚未被研究。此外，现有技术均未涉及内在码字不一致性问题，也未实际利用内在码字冗余。因此，我们首先推导出一种新的Hadamard码字解码过程，以获得最优类别概率，并通过投影到概率单纯形上来求解底层优化问题。其次，我们的优化提供了一种预测不一致性度量。第三，我们首次展示了如何利用这些不一致性进行对抗攻击与扰动检测。第四，我们引入了HadamardNet框架，该框架将Hadamard编码用作语义分割和目标检测模型及任务的输出表示。我们在扰动和对抗攻击上进行了全面评估，在仅一次检测过程中就实现了两个任务最先进的扰动检测性能，同时在干净数据上提供了相当或接近的参考性能。

## Abstract
Conventional one-hot encodings often yield poorly calibrated models, being overconfident under attack, and letting entropy-based detection algorithms fail. Previous image classification works have demonstrated that Hadamard-coded output representations can improve adversarial robustness. However, attempts to integrate Hadamard codes into semantic segmentation fall far behind state-of-the-art models in mean intersection-over-union performance. Regarding object detection, such output encodings have not yet been investigated at all. Further, no prior art addressed intrinsic codeword inconsistencies or actually exploited intrinsic codeword redundancy. Accordingly, we first derive a novel decoding procedure for Hadamard codewords towards optimal class-wise probabilities, solving the underlying optimization problem by using the projection onto the probability simplex. Second, our optimization delivers a measure of prediction inconsistency. Third, we are the first to show how to exploit these inconsistencies for adversarial attack and disturbance detection. Fourth, we introduce HadamardNet, a framework employing Hadamard codes as output representations for semantic segmentation and object detection models and tasks. We conduct a comprehensive evaluation both on disturbances and adversarial attacks, achieving state-of-the-art perturbation detection performance for both tasks in only a single detection pass, while delivering equivalent or close-by reference performance on clean data.

### 2. 论文提出的方法论

论文提出了一个名为 **HadamardNet** 的统一框架，其核心思想是利用纠错输出码（特别是 Hadamard 码）作为网络输出表示，并通过一种新的解码过程来获取扰动敏感信号。

*   **核心思想**：为每个类别分配一个唯一的、具有高且等距汉明距离的 Hadamard 码字作为目标标签，取代传统的独热向量。网络为每个像素/物体预测一个“软码字”。当输入被扰动时，该预测码字内部会出现不一致性，通过解码过程可以提取出这种不一致性作为检测信号。

*   **关键技术细节与算法流程**：
    *   **Hadamard 编码 (3.1节)**：使用 Sylvester 构造法生成 `L x S` 的 Hadamard 矩阵 `H`，其中 `L ≥ S`。每个列向量 `H_s` 即为类别 `s` 的码字。
    *   **HadamardNet 架构 (图1, 3.2节)**：网络最后一层后接 Sigmoid 激活函数，输出一个 `L` 维的软码字 `h`（每个元素在 [0, 1] 之间）。随后，通过一个**Hadamard 解码器**（核心创新）处理。
    *   **Hadamard 解码器优化问题 (核心创新)**：
        1.  **问题建模**：将软码字 `h` 与理想类别后验概率 `P` 的关系建模为一个线性方程组 `H * P = h`。由于预测的 `h` 存在矛盾，该方程组通常不可解。
        2.  **引入误差**：引入一个误差向量 `e`，将方程组变为 `H * P = h + e`，并寻求在概率约束下（和为1，非负）误差 `e` 的 L2 范数最小的解。
        3.  **求解与解码**：推导证明该优化问题等价于在概率单纯形上寻找最优投影。输出结果有两个：**(1) 最优类别后验概率 `P*`** (通过投影算法得到) 和 **(2) 误差向量 `e*`** (`e* = P* - P̃`)，后者衡量了预测不一致性。
    *   **扰动检测 (3.3节)**：基于解码器的输出 `P*` 和 `e*`，为每幅输入图像 `x` 计算一个异常分数 `δ(x)`，并与阈值比较。
        *   **误差向量异常分数**：直接使用平均 L1 范数 `δ(x) = avg(||e*_i||_1)`。
        *   **回归异常分数**：训练一个小的回归网络 `F`，输入 (最大后验概率 `P*_max`, 预测熵 `H(P*)`)，期望输出对应的 `||e*_i||_1`。异常分数是整个图像上预测误差的绝对偏差。
        *   **分位数异常分数**：训练回归网络 `F` 来预测 `||e*_i||_1` 的 85% 条件分位数。异常分数是实际误差超过该分位数的像素/物体比例。
    *   **训练 (3.4节)**：HadamardNet 的损失函数是编码空间损失 `J_ENC` (如 MSE)、解码空间损失 `J_DEC` (如交叉熵) 和鼓励码字一致性的误差正则项 `J_error` (最小化 `||e*_i||_2`) 的线性组合。扰动检测回归模型 `F` **仅在干净数据集上**单独训练。

1.  **统一的框架**：HadamardNet 成功地将 Hadamard 输出编码扩展到目标检测，并显著提升了其在语义分割上的性能，使其在干净数据上达到了与独热编码基线的同等或接近水平。

*   **方法论创新**：首次从数学上严格利用 Hadamard 码冗余，提出概率单纯形投影解码方法，并成功将其应用于**分割和检测两大核心视觉任务的攻击检测中**。

### 8. 不足与局限

*   **干净数据上的微小性能下降**：在某些配置下，HadamardNet 在干净数据上的 mIoU 或 AP 微低于独热编码基线（尽管作者称“接近”或“相当”），这可能成为对精度极其敏感的应用的障碍。

*   **方法局限性**：目标检测中的“消失攻击”由于是将所有物体更确信地预测为背景，减少了可用于检测的对象级不一致性，因此所有方法对此攻击的检测效果均不佳，本文方法也不例外，这是任务本身带来的固有挑战。

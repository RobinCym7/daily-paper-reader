## 摘要
视觉变换器（ViT）已成为视觉表征学习的主流架构，提供了异常强大且广泛可复用的骨干特征。然而，由于全局自注意力的平方代价，ViT 通常在相对较小的块令牌网格上运行，这为语义分割和深度估计等密集预测任务造成了持续瓶颈。这推动了任务无关特征上采样器的发展。尽管近期最先进方法能生成视觉上清晰的密集表示，但它们依赖浅层图像编码器进行引导上采样，可能引入特征泄漏、碎片化和模糊。我们提出 ViT-Up，一种隐式特征上采样框架，通过从中间 ViT 隐藏状态构建逐层查询来替代外部图像引导。这能在任意连续图像坐标上预测特征，同时保持与骨干特征空间的对齐。实验表明，ViT-Up 在密集预测和语义对应任务上始终优于最先进的图像引导上采样器。在 DINOv3-S+ 上，ViT-Up 在 Cityscapes 上比先前方法提升高达 +2.07 mIoU，在 SPair-71k 上提升 +4.17 PCK@0.10。使用更大的 DINOv3-B 骨干时，增益增至 +3.36 mIoU 和 +8.09 PCK@0.10，说明 ViT-Up 能随骨干容量良好扩展。

## Abstract
Vision Transformers (ViTs) have become a dominant architecture for visual representation learning, providing exceptionally strong and broadly reusable backbone features. However, ViTs are commonly operated on relatively small patch-token grids due to the quadratic cost of global self-attention, which creates a persistent bottleneck for dense prediction tasks such as semantic segmentation and depth estimation. This has motivated the development of task-agnostic feature upsamplers. While recent state-of-the-art methods produce visually sharp dense representations, their reliance on shallow image encoders for guided upsampling can introduce feature leakage, fragmentation, and blur. We introduce ViT-Up, an implicit feature upsampling framework that replaces external image guidance with layer-wise query construction from intermediate ViT hidden states. This enables feature prediction at arbitrary continuous image coordinates while preserving alignment with the backbone feature space. Experiments demonstrate that ViT-Up consistently outperforms state-of-the-art image-guided upsamplers across dense prediction and semantic correspondence. On DINOv3-S+, ViT-Up improves over prior methods by up to +2.07 mIoU on Cityscapes and +4.17 PCK@0.10 on SPair-71k. With the larger DINOv3-B backbone, these gains increase to +3.36 mIoU and +8.09 PCK@0.10, demonstrating that ViT-Up scales favorably with backbone capacity.

## 1. 论文的核心问题与整体含义

- **核心问题**：Vision Transformer 在特征表示上极为强大，但其全局自注意力计算复杂度随 token 数量呈平方增长，迫使模型只能在较粗的 patch 网格（如 14×14 或 28×28）上提取特征。这种低分辨率特征图成为语义分割、深度估计等密集预测任务的严重瓶颈。
- **现有方法局限**：近期任务无关的特征上采样器（如 JAFAR、AnyUp、UpLiFT、NAF）普遍依赖一个额外的浅层图像编码器作为高分辨率引导信号。这种做法虽然能产生视觉上清晰的密集特征，但浅层编码器缺乏足够的语义理解能力，容易导致 **特征泄漏**、**语义碎片化** 和 **模糊**——表面上的锐利掩盖了深层语义错乱。
- **整体含义**：论文提出 **ViT‑Up**，主张 **彻底抛弃外部图像引导**，转而利用 ViT 自身中间层的层次化隐藏状态，通过逐层构建查询来实现隐式特征上采样。这种方法能在任意连续图像坐标上预测保真度更高的密集特征，并保持与骨干特征空间的对齐，从而为各类下游密集预测任务提供更可靠的高分辨率特征。

## 2. 论文提出的方法论

### 2.1 核心思想
ViT‑Up 将特征上采样重新定义为 **坐标条件化的隐式特征解码** 问题：给定查询坐标 \( x_q \)，直接利用 ViT 骨干的中间隐藏状态逐层精炼出该位置的最终特征向量，而非像之前工作那样通过图像引导对低分辨率 token 进行重聚合。

### 2.2 关键技术与流程

**A. 查询初始化 Query Embedding**
- 将输入图像用 ViT 骨干的 patch embedding 层重新编码，得到 **高分辨率（如 224×224）** 的 patch‑token 网格。
- 在查询坐标 \( x_q \) 处对该网格进行双线性插值，得到初始查询表示 \( q_0 \)。复用 backbone 的 patch 嵌入，保证初始表示与骨干特征空间对齐。

**B. ViT‑Up 块（逐层精炼）**
\( q_t = U_t(q_{t-1}, x_q, H_{l[t]}) \)，总计 \( T \) 个块，仅选取骨干的若干中间层（如第 2、4、6、8、10、12 层）。每块包含：

- **过渡 MLP**：对上游查询表示做残差变换，对齐当前层特征空间。
- **Cross‑window Multi‑head Attention（CW‑MHA）**：让查询与当前层低分辨率空间 token 做交叉窗口注意力，聚合 token 级上下文。查询和键使用 2D 旋转位置编码 RoPE（支持连续坐标），通过 NATTEN 加速。
- **FeatX 子 token 特征提取模块**：
  - 找到离 \( x_q \) 最近的 patch token \( h_{nn} \) 及其中心坐标 \( x_{nn} \)。
  - 计算相对偏移 \( \Delta x = (x_q - x_{nn})/p \) 并做正弦位置编码。
  - 用 MLP 预测 FiLM 参数 \( \gamma, \beta \)，对最近 token 特征进行调制（`(1+γ)⊙LN(h_nn)+β`），再由 MLP 提取查询专属的细粒度子 token 信息。
- **融合**：`xfused = x + x_attn + x_sub_token`，再经残差融合 MLP 得到 \( q_t \)。

**C. 解码器**
一个单层的 MLP + LayerNorm + 线性投影，将最后的 \( q_T \) 映射回 ViT 特征空间，得到输出特征向量。

**D. 骨干适应**
对 ViT 骨干的 patch embedding 及注意力投影层施加 LoRA 低秩适配，以极少量参数为特征上采样提供额外容量。

### 2.3 训练策略

- **多尺度特征监督（学生‑教师蒸馏）**
  - 教师：将同一图像分别以 224、448、896 分辨率送入冻结的原始 ViT，获得对应 token 网格（14×14、28×28、56×56）的特征作为监督目标。
  - 学生：输入为随机缩放到 \( s \times 448 \)（\( s\sim U(0.1,1.0) \)）并粘贴到固定 448×448 黑色画布的图像。对画布内可见区域采样 56×56 的查询网格进行隐式上采样，并在各精炼阶段使用 AvgPool 将预测特征缩放至与教师各尺度匹配，施加损失。
- **损失函数**
  - **归一化 L2 损失**：对每个特征向量按教师通道均值和标准差归一化后计算 L2 距离。
  - **余弦相似度损失**：鼓励方向对齐。
  - **关系 KL 损失**：计算预测与教师特征之间的成对余弦相似度矩阵，然后最小化其 softmax 概率的 KL 散度，以保持特征空间的关系结构。
  - 总损失为三层损失的加权和（文中权重均设为 1），平均到所有被监督的层和 token 分辨率。

- **训练配置**：ImageNet‑1K 数据集，训练 1 个 epoch，batch size 24，学习率 \(2\times10^{-4}\)，余弦退火。

## 6. 主要结论与发现

1. **无需图像引导即可实现更保真的上采样**：ViT‑Up 在 Cityscapes 上比最强 image‑guided 方法 (NAF/UpLiFT) 提升 +2.07 mIoU，在 SPair‑71k 上 PCK@0.10 提升 +4.17，表明视觉锐利度 ≠ 语义保真度。
2. **骨干容量可扩展性**：当骨干从 S+ 扩至 B 时，ViT‑Up 的优势进一步扩大（Cityscapes +3.36 mIoU，SPair +8.09 PCK@0.1），而 image‑guided 方法因外部编码器能力固定而受限甚至性能下降。
3. **细粒度对应能力突出**：ViT‑Up 在 SPair‑71k 的严格阈值下几乎将双线性插值性能翻倍，证明其恢复了更精确的局部部件特征。
4. **特征空间兼容性好**：在冻结头测试中，ViT‑Up 在 VOC 和 Cityscapes 上远超各 image‑guided 方法，说明上采样后特征仍与低分辨率特征空间高度对齐。
5. **多尺度监督与层次精炼有效**：消融证实，逐层精炼（6 层明显优于 1‑2 层）、FeatX 子 token 提取、Cross‑Attention 等组件缺一不可，共同保障语义连贯性和空间细节。

## 7. 优点

- **架构创新**：首次将 ViT 中间隐藏状态的层次结构系统性地用于无图像引导的隐式坐标上采样，设计直观且与骨干特征空间高度对齐。
- **模块设计严谨**：FeatX 利用相对位置编码与 FiLM 调制，精细恢复子 patch 细节；Cross‑window attention 结合 RoPE 保持空间一致性；过渡 MLP 和融合策略让层间信息流动平滑。
- **训练策略巧妙**：多尺度教师特征+可缩放画布的设计，避免了直接在高分辨率下运行骨干导致开销大或特征退化的问题。
- **消融充分**：对核心组件、层数、输出分辨率都进行了定量分析，并通过可视化定性说明了各模块的作用。
- **可扩展且推理可控**：支持查询分块控制显存；在大模型上优势不减反增；输出分辨率可灵活调整。
- **实验全面且基准强**：覆盖分割、深度、语义/几何对应多个维度，且与最新 SOTA 在统一设置下对比，结论可信度高。

## 8. 不足与局限

- **信息瓶颈依然存在**：上采样质量最终受限于骨干隐藏状态包含的信息，无法恢复被骨干丢弃的细微结构。文中建议可将骨干分辨率适度提高至 56×56 来缓解。
- **骨干依赖性较强**：ViT‑Up 的过渡 MLP、FeatX 等需针对每种 ViT 分别训练，无法像 AnyUp/NAF 那样直接零样本迁移到新骨干。作者认为这是一次性开销，但限制了即插即用的便利性。
- **训练成本未完全透明**：虽说明了在 RTX 5090 上可训练，但未给出具体的 wall‑clock 时间或总 GPU 时，增加了复现难度。
- **与 artifact suppression 目标的冲突**：在像 DINOv2 这样带有明显位置伪影的骨干上，ViT‑Up 忠实地重建了这些伪影，导致在某些 task 上不如 image‑guided 方法的“去伪影”效果。因此适用范围更偏向现代干净特征（如 DINOv3）。
- **推理速度仍有优化空间**：虽然分块后显存友好，但在全分辨率 448×448 下比最快的 UpLiFT 慢，仅在降低查询分辨率（112×112）时能达到相近延迟。
- **未涉及时序/三维任务**：验证集中于图像级密集预测和对应，扩展到视频、3D 重建等场景的效果有待考察。

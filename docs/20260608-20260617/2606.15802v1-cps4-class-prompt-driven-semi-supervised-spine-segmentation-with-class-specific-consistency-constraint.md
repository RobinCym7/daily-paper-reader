## 摘要
视觉语言模型（VLM）通过利用文本类别提示生成分割图，在半监督脊柱分割中具有提升伪标签质量的巨大潜力，但尚未有人研究。尽管前景广阔，但缺乏显式约束来确保脊柱类别提示与脊柱单元区域之间的一致性，导致多类分割图生成性能不佳。在本文中，我们提出了CPS4，这是首个使用类别提示来提升脊柱伪标签质量的文本引导半监督脊柱分割网络。具体而言，CPS4通过两个训练阶段实现。(i) 类别特定一致性约束的VLM预训练阶段：我们提出词元级和像素级注意力损失，以优化类别提示与脊柱单元之间的一致性，迫使文本类别提示在语义空间中与目标脊柱单元紧密耦合。(ii) 类别提示驱动的半监督脊柱分割阶段：利用预训练的视觉-文本编码器，我们为无标注脊柱图像推导出每个类别特定的二值分割图，并将其整合为统一的多类分割图，从而提升半监督脊柱分割网络生成的脊柱伪标签质量。实验结果表明，我们的CPS4在公开脊柱分割数据集上仅使用5%标注数据，就实现了Dice为80.44%的优异脊柱分割性能，超越了流行的半监督学习和VLM方法。我们的代码将开源。

## Abstract
Vision Language Model (VLM) has great potential to enhance the quality of pseudo labels in semi-supervised spine segmentation by leveraging textual class prompts to generate segmentation map, but no one has studied it yet. Although promising, it lacks explicit constraints to ensure consistency between spine class prompts and spine unit region, resulting in unsatisfactory performance in multi-class segmentation map generation. In this paper, we propose CPS4, the first text-guided semi-supervised spine segmentation network using class prompts to enhance the quality of spine pseudo labels. Specifically, CPS4 is implemented through two training stages. (i) Class-specific consistency constrained VLM pretraining stage: we propose token- and pixel-level attention loss to optimize the consistency between class prompts and spine units, forcing the textual class prompt to be closely coupled with the target spine unit in the semantic space. (ii) Class Prompt driven semi-supervised spine segmentation stage: using the pretrained vision-text encoder, we derive each class-specific binary segmentation map for the unlabeled spine image and integrate them into an unified multi-class segmentation map, improving the quality of the spine pseudo label generated by the semi-supervised spine segmentation network. Experimental results show that our CPS4 achieves superior spine segmentation performance with Dice of 80.44%, only using 5% labeled data on the public spine segmentation dataset, surpassing popular semi-supervised learning and VLM methods. Our code will be available.

- **研究背景**：脊柱磁共振（MR）图像中椎体与椎间盘的自动多分类分割对脊柱疾病诊断、手术规划至关重要。现有医学图像分割依赖大量像素级标注，但脊柱图像多类别标注耗时费力且需专业经验。
- **问题提出**：半监督学习（SSL）利用少量标注和大量无标注数据缓解标注稀缺，但常因伪标签质量低而产生噪声监督，误导学习。视觉语言模型（VLM）可通过文本类别提示增强伪标签质量，但在多类脊柱分割中，现有 VLM 缺少**类别特定一致性约束**，导致文本提示与对应脊柱区域语义耦合弱、跨类干扰严重，难以生成准确的分割图。
- **核心含义**：本工作首次提出 **CPS⁴**——类别提示驱动的半监督脊柱分割网络，通过显式约束类别提示与脊柱单元的一致性，提升伪标签质量，实现高效的半监督多类分割。

## 2. 方法论：核心思想与技术细节

### 2.1 两阶段训练整体框架
CPS⁴ 由两个阶段构成：
1. **类别特定一致性约束的 VLM 预训练阶段**：利用词元级和像素级注意力损失加强类别提示与对应脊柱区域的对齐。
2. **类别提示驱动的半监督分割阶段**：利用预训练的视觉–文本编码器生成各类别二值分割图，融合后增强伪标签质量，再与均值教师（Mean Teacher）框架联合训练分割网络。

### 2.2 阶段一：类别特定一致性约束 VLM 预训练
- **目标**：弥补现有 VLM（如 CLIP、MedCLIP）仅做图像级对齐的不足，使每个类别提示能精准聚焦于图像中对应的脊柱单元。
- **词元级注意力损失 \(L_{token}\)**：
  - 对每类 \(i\)，从文本编码器提取类别提示词元 \(F_{T,i}\)，从视觉编码器提取图像补丁词元 \(F_I\)。
  - 计算相似度获得粗注意力图 \(A_i\)，并与该类别的真实二值分割图 \(M_i\)（下采样并二值化后记为 \(\tilde{M}_i\)）对齐。
  - 损失惩罚注意力图中脊柱单元区域内的激活总和占比偏离 1，迫使模型将注意力集中到正确脊柱区域：
    \[
    L_{token} = \frac{1}{N} \sum_i \left(1 - \frac{\sum_{u \in B_i} A_{(i,u)}}{\sum_u A_{(i,u)}}\right)^2
    \]
    其中 \(B_i\) 为 \(\tilde{M}_i\) 中值为 1 的脊柱单元像素集。
- **像素级注意力损失 \(L_{pixel}\)**：
  - 避免注意力过度集中到脊柱单元的子区域，促进激活均匀覆盖整个目标单元。
  - 采用二值交叉熵，使注意力图 \(A_i\) 逼近 \(\tilde{M}_i\)：
    \[
    L_{pixel} = -\frac{1}{N} \sum_i \left[ \tilde{M}_{(i,u)} \log A_{(i,u)} + (1 - \tilde{M}_{(i,u)}) \log(1 - A_{(i,u)}) \right]
    \]
- **预训练优化目标**：\(\frac{1}{2}(L_{token} + L_{pixel})\)

### 2.3 阶段二：类别提示驱动的半监督分割
- **类别提示引导的分割图生成**：
  - 对无标注图像，利用预训练视觉和文本编码器，计算各补丁词元与每个类别提示词元的点积，生成注意力图。
  - 上采样后，通过大津阈值法（OTSU）得到每类的二值分割图 \(y_i^{text}\)，堆叠形成最终的多类分割图 \(y^{text}\)。
  \[
  y_i^{text} = \text{OTSU}(\text{Up}(F_I \otimes F_{T,i}^\top)), \quad y^{text} = \text{stack}_{i \in N}(y_i^{text})
  \]
  值得注意的是，此过程无需预先知晓每张未标注图像中包含哪些类别，而是一次性生成所有类别（20类）的注意力图。
- **伪标签生成及增强**：
  - 基础 SSL 采用均值教师架构，包含学生模型 \(f_{\theta_s}\) 和教师模型 \(f_{\theta_t}\)（EMA 更新）。
  - 教师模型生成无标注图像的伪标签 \(y_t^u\)。
  - 利用 \(y^{text}\) 增强伪标签：计算学生预测与 \(y^{text}\) 之间的损失 \(L_{semi}^{text}\)，以及学生预测与增强伪标签（\(y_t^u + y^{text}\)）之间的损失 \(L_{semi}^{enhance}\)（实际公式里对增强伪标签求和）。
  - 总优化目标：\(\frac{1}{2}(L_{semi}^{text} + L_{semi}^{enhance}) + L_{sup}\)，其中 \(L_{sup}\) 为有标注数据上的监督交叉熵损失。

- **评价指标**：平均 Dice 系数（mDice）和平均交并比（mIoU）。
- **标注比例设置**：5%、10%、25%、50% 的标注数据，其余作为无标注数据。
- **对比方法**：涵盖两类主流方法：
  - **纯半监督分割方法**：Mean Teacher (MT)、BCP、MC-Net、SS-Net、UCMT。

## 6. 主要结论与发现

- CPS⁴ 首次将文本类别提示引入半监督脊柱分割，通过两阶段训练显著提升伪标签质量。
- 在 **5% 标注数据** 下，CPS⁴ 优于所有对比方法，mDice 达 80.44%，比次优的 GraphCL（78.21%）高 2.23%，比纯 SSL 方法（如 MT 74.25%）提升约 6.19%。
- 词元级与像素级注意力损失共同作用，有效强化类别提示与脊柱单元的语义绑定，避免注意力聚合过度或不足。
- 随着标注比例增加（5%→50%），CPS⁴ 持续领先，且性能下降慢，数据鲁棒性强。
- 定性注意力图显示，CPS⁴ 的类别特定注意力高度集中于目标脊柱区域，类间干扰小，而 MedCLIP 注意力弥散、语义绑定弱。

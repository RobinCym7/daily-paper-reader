## 摘要
我们提出GOOSE-M2F，这是针对ICRA 2026年GOOSE 2D细粒度语义分割（FGSS）挑战赛对Mask2Former进行的任务特定适配。GOOSE基准涵盖非结构化户外地形中的64个细粒度类别，分布呈严重长尾，其中稀有类别每张图像仅占少于50个像素。我们通过三项针对性贡献扩展了Swin-Large Mask2Former基线：（1）200个对象查询以消除表征饱和；（2）结合ASPP-lite和CBAM双重注意力的特征细化模块（FRM）；以及（3）为稀有类别提供直接逐像素梯度的辅助监督头。多阶段训练策略结合了分布平衡损失、稀有类别复制粘贴增强、动态IoU感知重新加权和EMA。推理时，使用具有2D高斯核混合和4尺度TTA的密集滑动窗口引擎，性能提升+10.57%。GOOSE-M2F取得了70.08%的官方综合mIoU（精细63.55%，粗略76.61%），在GOOSE 2D FGSS排行榜上位列第三。代码和训练模型已公开提供：Github GOOSE-M2F Code 和 Hugging Face GOOSE-M2F。

## Abstract
We present GOOSE-M2F, a task-specific adaptation of Mask2Former for the GOOSE 2D Fine-Grained Semantic Segmentation (FGSS) Challenge at ICRA~2026. The GOOSE benchmark spans 64 fine-grained classes across unstructured outdoor terrain with a severely long-tailed distribution, where rare classes occupy fewer than 50 pixels per image. We extend the Swin-Large Mask2Former baseline with three targeted contributions: (1)200 Object Queries to eliminate representational saturation; (2)a Feature Refinement Module (FRM) combining ASPP-lite and CBAM dual-attention; and (3)an Auxiliary Supervision Head that delivers direct per-pixel gradients for rare classes. A multi-stage training strategy pairs Distribution-Balanced loss, Rare-Class Copy-Paste augmentation, dynamic IoU-aware re-weighting, and EMA. At inference, a dense sliding-window engine with 2D Gaussian kernel blending and 4-scale TTA adds +10.57\%. GOOSE-M2F achieves 70.08\% Official Composite mIoU (63.55\% fine, 76.61\% coarse), placing 3rd on the GOOSE 2D FGSS leaderboard. Code and trained models are publicly available at: \href{https://github.com/Aditya-Lingam-9000/GOOSE-M2F}{Github GOOSE-M2F Code} and \href{https://huggingface.co/XYZ9843/GOOSE-M2F}{Hugging Face GOOSE-M2F}.

### 2. 方法论
论文基于 **Mask2Former（Swin-Large 骨干）** 进行任务特定适配，核心思想是**消除查询饱和、增强多尺度特征、为稀有类提供直接监督**，并辅以长尾训练与融合推理。

**关键技术贡献：**
- **200 个目标查询**：将默认的 100 个查询扩展到 200 个，新增的 100 个查询以原预训练嵌入为中心、加入微小高斯扰动 `qᵢ = qᵢ mod ₁₀₀ + εᵢ, εᵢ ~ N(0, 0.02²I)`，从而扩大解码器容量，解决 64 类场景中 15-20 个语义区域造成的表示饱和问题。
- **特征细化模块（FRM）**：置于像素解码器与 Transformer 解码器之间，解决单尺度 `H/4` 特征的局限。包含：
  - **ASPP-lite**：并行空洞卷积（膨胀率 `d ∈ {1,3,6,12}`）加全局池化，经 1×1 卷积融合，不同膨胀率分别对应纤小物体、中等物体、大面积模糊区域和场景上下文。
  - **CBAM 双重注意力**：先通道注意力，再空间注意力，并加入残差连接 `x + F_fused` 保持梯度。
- **辅助逐像素监督头**：在训练时直接对 `H/4` 特征图施加**逐像素交叉熵损失**，避开匈牙利匹配对稀有类的忽略，为稀有类提供密集梯度信号，推理时移除以节省资源。总损失为 `L_total = L_M2F + 0.4 L_aux`。
- **长尾训练策略组合**：
  - **分布平衡损失**：类别权重 `w_c = (1-β)/(1-β^{n_c})`，β=0.9999，裁剪至 [0.1, 10.0]。
  - **动态 IoU 重加权**：每轮验证后，根据 EMA 模型的各类别 IoU 动态调整损失乘数（IoU 低的类乘数高）。
  - **稀有类别复制粘贴增强**：预提取的稀有类小块以 0.85 概率、通过 5×5 高斯软掩膜融合到训练图像中，微小目标额外缩放。
  - **类别感知重复采样**：根据图像中稀有类的频率过采样图像，最高 3 倍。
  - **EMA**：影子权重动量 0.9995，用于所有验证和最终提交。
- **推理引擎**：对高分辨率图像使用**密集滑动窗口（896×896，步长 384，57%重合）**，窗口内预测用 **2D 高斯核加权融合**去缝；并结合**4 尺度（0.5×, 0.75×, 1.0×, 1.5×）× 水平翻转的测试时增强（TTA）**，8 个视图的 logits 平均后取 argmax。
